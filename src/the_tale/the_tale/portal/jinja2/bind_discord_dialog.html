Связывание аккаунтов игры и Discord

Чтобы привязать свой аккаунт к Discord, введите следующее сообщение в личной переписке с нашим ботом на сервере. Имя бота: «{{settings.DISCORD_BOT_NAME}}».

{% set command = settings.DISCORD_BIND_COMMAND + ' ' + bind_code.code.hex %} {{command}}   Копировать

Обратите внимание:

  • Команда активна {{expire_timeout|verbose_timedelta}}.
  • Код связывания аккаунтов однаразовый.
  • Если вы откроете этот диалог повторно, код связывания изменится и его необходимо будет скопировать заново.
  • К одному аккаунту Discord можно привязать только один аккаунт игры (и наоборот).
